Type of Machine Learning







There are many types of Machine Learning systems. That's is why these systems are classified in the following categories:

Supervised, Unsupervised, Semisupervised and Reinforcement Learning

Is the system trained with human supervision?

Supervised Learning

In this case the training set for the algorithm includes the desired solutions. These solutions are called label.

Unsupervised Learning

Unsupervised learning uses only unlabelled data. The systems learn on its own and tries to find a pattern.

Semisupervised Learning

Semisupervised learning works with both labeled and unlabeled data.

Reinforcement Learning

In this context the learning system observes the environment, selects an action and performs this action. Afterwards the system get a reward or a penalty depend on the action. Thus, this system can learn on its own what is the best strategy.

Online and Batch Learning

Can the system learn incrementally on the fly?

Online Learning

The system learns incrementally by giving it new data constantly. The system is able to adapt easily and fast.

Batch Learning

In this context the system is unable to learn incrementally. First you have to train the system with the training set and afterwards you can use the test set to see how the system performs. Using the test set the system doesn't learn anymore.
To improve the system you have to create a new version of the existing system and feed it with new training data. The new system replaces the old system.

Instance-based and Model-based Learning

Does the system work by simply comparing new data points to known data points or instead by detecting patterns in the training data and building a predictive model?

Instance-based Learning

The system produces a class label or a prediction based on the similarity of the nearest data in the training set. Instance-based learning uses the solutions of similar past problems to solve new problems.

Model-based Learning

A model is built out of a set of examples. Then this model will be used to make predictions about a set of new data.
