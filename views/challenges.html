Main challenges of Machine Learning


Machine Learning is about selecting a learning algorithm and training it on data.
So what can possibly go wrong?
The answer is: bad algorithm and bad data

BAD DATA

Insufficient Quantity of Training Data

Most algorithms need huge amounts of data to work properly

Nonrepresentative Training Data

Your algorithm is less likely to perform well if your data is not representative of the new cases you want to apply your algorithm to

Poor-Quality Data

For the algorithm it is hard to detect the underlying patterns in the data if it is full of outliers, errors and noise which makes the algorithm less likely to perform well

Irrelevant Features

The data has to contain enough relevant features in order for the system to be capable of learning, which can be avoided by coming up with a good set of features to train on

BAD
ALGORITHM

Overfitting the Training Data

The model performs well on the training data but is too flexible, so it does not generalize well

Underfitting the Training Data

The model is too simple to learn the underlying patterns in the data
