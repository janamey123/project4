but it's easy for machine learning tools to find hidden insights."

What is Machine Learning?

Basically, Machine Learning is finding patterns in huge amounts of data and applying them to new data.

A Machine Learning algorithm can find patterns in data by using statistics. The data can be anything that can be stored digitally: Pictures, texts, audio, ...

After training the algorithm with huge amounts of data, the pattern the algorithm has found can be applied to new data.

Where to apply Machine Learning?

One of many areas where Machine Learning is applied are recommendation systems on platforms like Netflix and YouTube.

These platforms collect data about you and try to find pattern within that data. The goal is to e. g. give you recommendations on what to watch next by making a highly educated guess based on the patterns found in your data.

Other applications of Machine Learning can be search engines or voice assistants. There are endless opportunities for using Machine Learning.
